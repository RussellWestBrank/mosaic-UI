# 快速上手

# 使用
在入口文件加入css样式:
```css
  html {
    --button-height: 32px;
    --font-size: 14px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666;
  }
```
在组件里引用:
```js
 import {Button} from "@mosaic-huojing"
```
